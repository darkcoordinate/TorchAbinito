$$\hat{H} |\Psi \rangle = E|\Psi \rangle$$

$$\left [ \frac{-\hbar}{2m} \nabla^{2}  + V(\mathbf{r})\right ] \Psi (\mathbf{r}) = E\Psi (\mathbf{r})$$
